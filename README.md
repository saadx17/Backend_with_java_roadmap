# Interesting fact on ReadMe
Early READMEs were crucial for explaining how to install software or giving instructions, as printed manuals were often slow to distribute. The name likely originated in the early 1970s on Unix systems and the Incompatible Timesharing System (ITS) at MIT, where they were often named `READ-ME` or `READ ME` to appear at the top of file listings. Some suggest the name was inspired by the "Eat Me" or "Drink Me" labels in Lewis Carroll's _Alice's Adventures in Wonderland_, serving as a playful command to the user to read the file first.
